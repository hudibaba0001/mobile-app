# KvikTime Localization Glossary

This document defines the official terminology for translations.
All translators must use these terms consistently.

## Core Concepts

| English | Swedish | Notes |
|---------|---------|-------|
| Time balance | Flexsaldo | Main balance term |
| Starting balance | Startsaldo | Opening balance at tracking start |
| Opening balance | Ingående saldo | Alternative, less common |
| Variance | Avvikelse | Difference from scheduled |

| Actual hours | Arbetad tid | Time actually worked |
| Credited hours | Tillgodoräknad tid | From paid absences |

## Key Naming Convention

Use these prefixes for ARB keys:
- `settings_` - Settings screen
- `contract_` - Contract settings
- `balance_` - Time balance related
- `leave_` - Leaves/absences
- `adjustment_` - Balance adjustments
- `redDay_` - Red days/holidays
- `export_` - Export/share
- `common_` - Shared/common strings
- `error_` - Error messages
- `action_` - Action buttons
